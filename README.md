# Тема uchebochka
Кастомная тема WordPress для витрины маркетплейса методических материалов. Использует Carbon Fields и интеграции с Ultimate Member/Yoast.

## Быстрый обзор
- Рендеринг: базовые шаблоны Wordpress + шаблонные функции в inc/.
- Контент: страницы `front-page.php`, `page-*.php`, `single.php`, `archive*.php`, таксономии `metodic_category` и CPT `metodic_post`.
- UI: компонентный подход через template-parts/; стили и скрипты собираются из src/.

## Каталоги и назначение
| Путь | Назначение |
| --- | --- |
| inc/ | Логика темы: инициализация, шаблонные функции, интеграции UM/Yoast, форматирование, таксономии, рейтинги. |
| template-parts/ | Повторно используемые фрагменты (каталог, фильтры, профили, рефералы, формы создания материалов, пагинация и т.д.). |
| src/css/ | Исходники стилей. |
| src/js/ | Исходники JS-модулей, собираются в итоговые бандлы. |
| img/ | Изображения и иконки темы. |
| lib/ | Локальные JS-библиотеки/плагины. |
| languages/ | Файлы переводов темы. |
| amp/ | Шаблоны и ресурсы AMP/ PWA (index.html, manifest, sw.js). |
| functions.php, front-page.php, page*.php, single*.php, archive*.php | Ключевые шаблоны и точки входа вывода. |

## Принципы разработки
- Соблюдайте структуру: бизнес-логика остается в плагине, тема отвечает за вывод.
- Используйте готовые helpers из inc/ вместо дублирования логики.
- Изменения в разметке оформляйте через template-parts/ для повторного использования.
- Следуйте WordPress coding standards; сохраняйте именование и префиксы темы.
